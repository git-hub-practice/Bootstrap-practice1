    Installation

    Include Bootstrap’s source Sass and JavaScript files
    via npm, Composer or Meteor. Package managed
    installs don’t include documentation, but do include
    our build system and readme.


    $ npm install bootstrap


    $ gem install bootstrap -v 4.3.1
